Site Plan: MealMatch

Site Name

MealMatch: Your Ingredient-Based Recipe Finder

This name was selected because it is going to be the funtionality of the website - matching ingredients to recipes. The name "MealMatch" makes it catchy, while the subtitle directly explains the porpouse of the website to the users. The name shows a helpful tool that solves the common problem of figuring out what to cook with available ingredients.

Site Purpose

MealMatch serves as an interactive tool designed to help users discover recipes based on ingredients they already have in their kitchen. The site will:

  • Allow users to input ingredients they have on hand and receive relevant recipe suggestions
  • Provide complete recipe information including ingredients, instructions, and images
  • Help reduce food waste by encouraging the use of available ingredients
  • Save users' recent searches to improve the user experience
  • Offer a submission form for user feedback and recipe suggestions

The primary goal is to simplify meal planning and preparation by bridging the gap between available ingredients and easy recipes, making cooking more accessible and reducing the stress of "what to cook".

Scenarios

Scenario 1

"I have chicken, bell peppers, and onions in my refrigerator but I'm not sure what to make with them. What recipes can I create with these ingredients?"

Scenario 2

"I'm trying to use up ingredients before they expire. Can I find recipes that specifically use the items I need to consume soon?"

Scenario 3

"I found a great recipe last week using the site, but I forgot to save it. How can I access my search history to find it again?"

Color Schema

#E63946

Primary Color

A warm red that shows appetite. Will be used for primary buttons, important headings, and call-to-action elements.

#F1FAEE

Secondary Color

A soft cream color that provides a clean background. Will be used for page backgrounds and content areas.

#1D3557

Text Color

A deep blue that provides strong contrast for readability. Will be used for body text, secondary headings, and navigation elements.

#457B9D

Accent Color

A medium blue-green that complements the primary color. Will be used for secondary buttons, highlights, and interactive elements.

Typography

Montserrat

This is a sample of the Montserrat font.

Usage: Will be used for main headings, navigation items, and buttons. Its clean, modern appearance provides good readability for important elements while conveying a contemporary feel.

Open Sans

This is a sample of the Open Sans font.

Usage: Will be used for body text, recipe instructions, and ingredient lists. Its high readability makes it perfect for longer text sections and detailed information.

Playfair Display

This is a sample of the Playfair Display font.

Usage: Will be used sparingly for recipe titles, quotes, and special sections. Its elegant style adds a touch of sophistication and helps highlight featured content.
